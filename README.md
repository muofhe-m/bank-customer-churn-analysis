# Bank Customer Churn Analysis

## Project Overview

This project analyzes a bank customer churn dataset using Python to identify the factors that influence customers leaving the bank. Through data cleaning, exploratory data analysis (EDA), and visualization, I explored customer behaviour and provided business recommendations to help improve customer retention.

This project is part of my data analytics portfolio and demonstrates my ability to clean data, analyze trends, create visualizations, and communicate business insights.

# Dataset

The dataset contains information about 10,000 bank customers, including:

- Credit Score
- Geography
- Gender
- Age
- Tenure
- Account Balance
- Number of Products
- Credit Card Ownership
- Active Membership Status
- Estimated Salary
- Customer Churn

**Target Variable**

- **0** = Customer Stayed
- **1** = Customer Left (Churned)

# Key Findings

- Approximately **20%** of customers in the dataset left the bank.
- Older customers showed a higher churn rate than younger customers.
- Churn rates varied across different countries.
- Inactive customers were significantly more likely to leave the bank.
- Customers with fewer banking products had higher churn rates.
- Customers who left the bank generally had higher account balances.
- Gender showed a slight difference in churn rates, but further analysis would be needed to understand the reasons behind this difference.

---

# Business Recommendations

Based on the analysis, I recommend that the bank:

- Improve customer engagement for inactive members.
- Encourage customers to use more banking products to increase loyalty.
- Develop retention strategies for customers with high account balances.
- Investigate regional differences in churn to create targeted solutions.
- Continue monitoring customer behaviour to identify customers at risk of leaving.
